# Pandoc image

## Overview
Docker image including Pandoc and dependencies to support additional formats (installed from the Debian package).

### Included package
* pandoc
* texlive-latex-recommended
* texlive-latex-extra
* require texlive-xetex
* texlive-luatex
* librsvg2-bin
* context
* groff
* libjs-mathjax
